## Overview

This site. Built to stand out from the typical portfolio template — it opens with a fake terminal "system takeover" sequence personalized with the visitor's name, blasts static noise, drops a "jk", then loads the actual portfolio.

## What I Built

- Jekyll static site generator with GitHub Pages hosting
- Full JavaScript terminal intro sequence with frame-by-frame canvas static noise
- Scroll-driven landing page with smooth section transitions
- Responsive layout with Bootstrap grid and custom dark design system

## Tech Stack

- **Jekyll** — static site generator
- **GitHub Pages** — hosting
- **Bootstrap** — responsive grid
- **Vanilla JavaScript + Canvas API** — intro sequence
